# CH32V208_dome 验证

## 项目概述

本项目是 CH32V208 的学习记录和驱动示例。该验证板集成了多种外设和传感器，用于学习和验证 CH32V208 的各项功能。

## 硬件资源

验证板搭载的主要硬件资源包括：

- **主控芯片**: CH32V208gbu6/wbu6
- **板载设备**:
  - 2.4G射频模块（型号: si24r1 ）
  - oled屏幕（型号: i2c屏 ）
- **其他外设**:
  - LED 指示灯
  - 按键
